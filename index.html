🪄 C 포인터 쉽게 배우기

초등학생도 이해할 수 있는 5단계 온라인 강의

포인터는 C 언어에서 ‘어디에 저장되어 있는지’를 가리키는 도구예요. 우편함 번호처럼 ‘몇 번 방에 있는지’만 알면 그 안에 있는 값을 찾을 수 있어요.

아래 5단계를 순서대로 따라 하면 포인터를 쉽게 이해할 수 있어요. 각 단계마다 개념 설명, 📋 실습 과제(구체적인 문제와 참고 정답 코드), 🔬 심화 학습이 모두 그 페이지 안에 들어 있어요. 코드는 문법 하이라이트와 줄 번호로 읽기 쉽게 표시돼요.

✏️ 실습 코드창: 각 단계의 과제 바로 아래에 실습할 수 있는 코드 편집창이 있어요. 모바일에서도 앱을 왔다 갔다 하지 않고 같은 화면에서 코드를 쓰고, [포맷] 버튼으로 자동 정리·[복사] 버튼으로 복사할 수 있어요. 에디터에서 포커스를 잃을 때(다른 곳을 눌렀을 때) 자동 포맷(오토 포맷)도 적용돼요.

1단계 📬 주소가 뭐예요? 우편함 비유로 ‘주소’ 개념을 이해해요. 2단계 📦 변수와 주소 변수는 메모리 어딘가에 있고, 그곳에 ‘주소’가 있어요. 3단계 🎯 포인터 변수 주소를 저장하는 특별한 변수, 포인터를 배워요. 4단계 ✏️ 포인터로 값 바꾸기 포인터를 이용해서 다른 변수의 값을 바꿔요. 5단계 🚀 포인터 활용하기 배열, 함수에서 포인터를 어떻게 쓰는지 알아봐요.
각 단계의 코드는 페이지에 그대로 있어요. 복사해서 온라인 C 컴파일러에서 실행해 보세요. (iframe 없이 모든 내용이 해당 페이지에 임베드되어 있습니다.)
